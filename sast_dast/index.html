Running SAST Scans Locally

Backend (Python)

docker run --rm -v "$(pwd)":/src returntocorp/semgrep \
  semgrep scan --config=auto /src > semgrep_results.txt

Frontend (JavaScript/React)

docker run --rm -v "$(pwd)":/src returntocorp/semgrep \
  semgrep scan --config=auto /src > semgrep_results.txt
